# File Arranger Pro Desktop

A Python desktop utility to arrange, separate, and move files/folders directly on the user's filesystem. Supports folder selection, file classification, and moving files into organized folders. Requires Python 3.7+.

## Features
- Select a source folder
- Automatically classify files by type (images, docs, code, etc.)
- Move files into new organized folders
- Arranged folder named for recognition
- CLI and Tkinter GUI support

## Usage
1. Install dependencies: `pip install tk` (Tkinter is usually included)
2. Run: `python file_arranger_desktop.py`
